Ejercicio de practica utilizando Spring Boot y Jpa - Hibernate.

En éste ejercicio voy a realizar una API para la gestión de dos clases, Curso y Tema. Las mismas cumplen con una asociacon 1 - N , es decir, un curso puede tener muchos temas
En principio utilizaré las siguientes dependencias:
- Lombock
- Spring Boot DevTools
- Spring Web
- Spring Data JPA
- H2 DataBase
- MySQL Driver
